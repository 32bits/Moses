_.clone (obj)
Clones an object. This function features recursion.

Parameters

  • obj: an object

Usage:

 local t = {1, 2, {3}} 
 local t_clone = _.clone(t) 
 t[3][1] = 'banana' 
 t_clone[3][1] --> 3

Return value:

an object

_.result (obj, property, ...)
Invokes an object's method with custom arguments

Parameters

  • obj: an object
  • property: a method name to be called, passed as a string
  • ...: a list of extra arguments to be passed to the property being called

Usage:

 local vector = {x = 0, y = 0} 
 function vector:move(dx,dy) 
 self.x,self.y = self.x+dx,self.y+dy 
 end 
 _.result(vector,'move',5,10) --> vector { x = 5, y = 10 }

Return value:

the expected result from the call obj:property(...)
_.template (obj, template)
Transforms a table and makes it look like a given template

Parameters

  • obj: an object
  • template: a given template

Usage:

 local templateVector = {x = 0, y = 0} 
 local particle = {velocity = 10, mass = 1} 
 _.template(particle,templateVector) --> {velocity = 10, mass = 1, x = 0, y = 0}

Return value:

a table
