_.invoke (method, list, ...)
Calls a method over each element in a collection of objects

Parameters

  • method: a method name passed as a string
  • list: a collection of objects
  • ...: a list of extra arguments to be passed to the invoked method.

Usage:

 -- A simple Window class 
 local Window = {width = 0, height = 0} 
 Window.__index = Window 
 --A constructor for window class 
 function Window:new() 
 return setmetatable({},self) 
 end 
 --A method for window class 
 function Window:resize(x,y) 
 self.x,self.y = x,y 
 end 
 --Creating a collection of window objects 
 local wins = {} 
 for i = 1,10 do wins[i] = Window:new() end 
 _.invoke('resize',wins,50,30) 
 end

Return value:

the expected result of the call list[i]:method(...)
